# Ruff

```sh
# フォーマット差分確認
ruff format --check --diff --config ~/ruff.toml <target_name>

# フォーマット修正
ruff format --config ~/ruff.toml <target_name>

# I001ルールを指定してチェック
ruff check --select I001 <target_name>
```
