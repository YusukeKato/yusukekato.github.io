# How to use Git
- gitコマンドの使い方のメモ書き

# Config
```sh
git config --global user.name "name"
git config --global user.email "email"
```

# Update commit message
```sh
# コミット
git commit -m "old message"
# コミットメッセージを更新
git commit --amend -m "new message"
```

# Tag
```sh
git tag 1.0.0
git push --tags origin
```
